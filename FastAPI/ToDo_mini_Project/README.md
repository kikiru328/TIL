# FASTAPI : ToDo Mini Project

**Class**  : [실전! FastAPI 입문](https://www.inflearn.com/course/%EC%8B%A4%EC%A0%84-fastapi-%EC%9E%85%EB%AC%B8/dashboard)  
**Contents** : FastAPI를 활용하여 단순한 CRUD 서비스가 아닌, 실무 실습을 가정한 프로젝트 개발  

---

### 배운점
이전에는 Flask, Django를 간단히 사용하면서 Python Web Framework나 백엔드 환경에 대한 고려 없이 개발했다.  
개념 없이 바로 기능을 구현하려 하니 문제 해결도 어렵고 구현도 막막했다.  
이번 강의를 통해 FastAPI의 구조와 활용법을 익히고, 백엔드 환경 구성에 대한 감을 어느 정도 잡을 수 있었다.

### Comments  
복습 시 전체 서비스 구조의 흐름을 도식화하면서 따라가는 것이 효과적일 것으로 보인다.  
구성 요소 간 연결 방식이 명확히 보이지 않아 어려웠고, 추후에는 구조를 구체화하여 학습해야겠다.
