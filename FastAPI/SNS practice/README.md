# FASTAPI : SNS Practice Project

**Purpose** : Django 기반으로 프로젝트를 시작하기 전에, 기존에 학습했던 FastAPI를 활용하여 SNS 서비스를 직접 구성해보았다.

### 기능 요약

**Post (Thread)**  
- [X] 게시글 작성  
- [X] 전체 게시글 조회  
- [X] 사용자별 게시글 조회  
- [X] 게시글 수정  
- [X] 게시글 삭제  

**Like**  
- [X] 좋아요 등록  
- [X] 좋아요 취소  
- [X] 좋아요 수 조회  

**Comment**  
- [X] 댓글 작성  
- [X] 댓글 수정  
- [X] 게시글별 댓글 조회  

**Newsfeed**  
- [X] 전체 뉴스피드 조회  

---

이 프로젝트는 FastAPI 기반으로 구현되었으며,  
이후 Django 환경으로 마이그레이션하여 `django/Sns`로 이전되었다.
